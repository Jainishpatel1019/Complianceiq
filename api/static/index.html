How It Works
Four automated steps from raw rule text to executive briefing
1

Ingest

Airflow DAG polls the Federal Register API every 6 hours, fetching new rules across OCC, FDIC, FRB, CFPB, and FinCEN

2

Embed & Score

Text is chunked, embedded via sentence-transformers, then scored using TF-IDF cosine drift + Jensen-Shannon divergence + Wasserstein distance

3

Causal Analysis

For regulations above the drift threshold, a LangChain agent runs DiD causal models to estimate actual financial impact (ATT)

4

Alert & Act

Compliance officers receive ranked alerts with dollar estimates, diff previews, and step-by-step recommended actions
