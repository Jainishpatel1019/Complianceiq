How It Works
Four automated steps from raw rule text to executive briefing
1

Ingest

Airflow polls the Federal Register every 6 hours, fetching new rules across OCC, FDIC, FRB, CFPB, and FinCEN

2

Score

Text is chunked and scored using TF-IDF cosine drift + Jensen-Shannon divergence + Wasserstein distance

3

Causal Analysis

For high-drift regulations, a LangChain agent runs DiD causal models to estimate actual financial impact (ATT)

4

Alert

Compliance officers receive ranked alerts with dollar estimates, diff proofs, and step-by-step actions
