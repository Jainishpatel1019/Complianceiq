Technology Stack
Production-grade infrastructure powering ComplianceIQ
🧠
LangChain + Ollama
Local LLM pipeline for plain-English regulation summaries and impact estimation. Zero cloud API dependency.
🗄️
TimescaleDB (PostgreSQL)
Time-series optimised DB for regulation versions and drift score history. Hypertables for fast range queries.
🔍
ChromaDB
Vector store for semantic search across 500+ regulation embeddings. Cosine similarity retrieval.
⚡
FastAPI + asyncpg
Fully async REST API with typed Pydantic schemas, OpenAPI docs, and WebSocket support.
🔬
TF-IDF · JSD · Wasserstein
Three-method drift detection ensemble with bootstrap confidence intervals (60-sample, 95% CI).
🌊
Apache Airflow
Daily DAG fetches Federal Register publications at 07:00 UTC. Weekly 14-day catch-up on Mondays.
🐳
Docker + HuggingFace Spaces
Single-container deployment with Postgres + ChromaDB + Airflow + uvicorn. Cold-start in <90s.
📊
Federal Register API
Real-time ingestion from api.federalregister.gov covering all banking & finance rule types.
🔗
Alembic Migrations
Version-controlled schema with composite PKs for TimescaleDB hypertable compatibility.
